# PhyloClimb（仮）ゲーム企画書

## 1. 企画概要

**「食べて進化する登攀ローグライク」**。
プレイヤーは原初の球体生命として生まれ、フィールドに散らばる餌を食べることで身体が変化し、新たな移動手段を獲得していく。グラップルも跳躍も最初は持っていない——すべては「何を食べたか」で決まる。

**タイトル候補**: PhyloClimb
**ジャンル**: 食性進化アクション × ローグライク × 進化図鑑コレクション
**価格**: ¥490
**想定プレイ時間**: 初クリアまで3〜8時間（1ランあたり20〜40分）

---

## 2. 背景・目的

### 2.1 背景

- **「Getting Over It」系は配信映えするが、リトライ性が低く飽きやすい**
- **ローグライクは成長感があるが、「自分の選択で身体が変わる」直感的な体験は少ない**
- **「何を食べるかで進化が変わる」は直感的で、配信者が実況しやすいコンセプト**

### 2.2 目的

- **Steam商業販売での収益化**（個人開発プロジェクト）
- 配信者が「食べたら〇〇になった！」と盛り上がれる設計による自然な拡散
- 「毎ラン違う身体・違うルート」による高いリプレイ性

## 4. 現状の課題と解決策

### 4.1 既存ジャンルの課題

| 課題 | 影響 |
|------|------|
| **登攀ゲームの単調なリトライ** | 毎回同じ操作の繰り返しで飽きる |
| **ローグライクの成長が数値的** | 「+10%ダメージ」等は体感しにくい |
| **移動手段が最初から揃っている** | 成長の驚きが少ない |

### 4.2 本企画での解決策

| 課題 | 解決策 |
|------|--------|
| **単調なリトライ** | **食性で進化が変わる**：毎ラン異なる身体・異なる攻略法 |
| **数値的な成長** | **移動手段自体が進化で解禁**：ジャンプ・グラップル・滑空が「生える」 |
| **最初から揃っている** | **球体スタート**：左右移動のみから始め、食べるたびに世界が広がる |

## 5. 体験の柱（コアゲームデザイン）

### 5.1 5つの核心体験

1. **食べて変わる驚き**
   - 球体→手が生える→掴めるようになる、という直感的な成長
   - 「何を食べたか」で身体の形も能力も変わる

2. **移動手段が"進化で増える"気持ちよさ**
   - 最初は転がるだけ→ジャンプ解禁→グラップル解禁→滑空解禁
   - 新しい移動手段を得るたびに「行ける場所」が劇的に増える

3. **落下の緊張感 + 欠片散布の駆け引き**
   - 落下ダメージ＋自分の欠片がばら撒かれる
   - 回収すれば進化に使える、放置すれば敵が食べて強化される

4. **食性による戦略的ビルド**
   - Dust（安定）/ Sap（機動）/ Metal（耐久）/ Flesh（攻撃）の選択
   - 何を食べるか＝どう進化するか＝どのルートで攻略するか

5. **進化図鑑が埋まる快感**
   - 50+ノードの系統樹コレクション
   - 死んでも発見は永続。「この食べ方をするとこう進化するのか」

## 6. コアループ

```
1. フィールドを探索する（左右移動、獲得済みの移動手段を駆使）
   ↓
2. 餌を見つけて食べる（接触吸収）
   ↓
3. 摂取比率が閾値に達すると自動進化（新しい身体・能力を獲得）
   ↓
4. 新たな移動手段で行ける場所が増える
   ↓
5. 奥に進むほど良い餌がある（ただしリスクも増える）
   ├─ 敵が強い
   ├─ 落下で欠片を撒く→敵が食べて強化
   └─ 即死ゾーンがある
   ↓
6. 3つのゴールのいずれかに到達→クリア
   ↓
7. 死亡→進化リセット、ただし図鑑は永続記録
   ↓
8. 新たな食性戦略で再挑戦
```

## 7. 食性と進化システム

### 7.1 餌の4系統

プレイヤーの進化は「何を食べたか」の比率で自動的に決まる。プレイヤーが選ぶのは「どの餌を食べるか」であり、進化自体は食性比率に基づいて自動発生する。

| 食性 | 色 | 入手場所 | 進化の方向性 |
|------|----|----------|--------------|
| **Dust（塵）** | 灰白 | 序盤の安全地帯に豊富 | 安定・伸長・粘着（基盤系） |
| **Sap（樹液）** | 琥珀 | 高所の樹上・柱上 | 跳躍・牽引・滑空（機動系） |
| **Metal（鉄粉）** | 鉄青 | 洞窟・断崖の壁面 | 外殻・重量・耐久（防御系） |
| **Flesh（肉）** | 深紅 | 敵を追い払うとドロップ | 攻撃・捕食・威嚇（戦闘系） |

### 7.2 進化の発生ルール

- **閾値制**: 特定の食性の摂取量が閾値に達すると自動的に進化が発生
- **比率制分岐**: 複数の食性を混合摂取した場合、比率に応じて混合進化が発生
- **不可逆**: 進化は1ラン内で元に戻せない（食べたものは取り消せない）
- **ラン制リセット**: 死亡時に全進化がリセットされる

### 7.3 進化ツリー概要（50+ノード）

進化ツリーは食性ごとの大枝と、その中のサブ枝で構成される。

#### Dust系（塵）— 安定・基盤 （約12ノード）

```
Dust
├─ 伸長枝
│  ├─ 伸長I（体が少し伸びる→段差に引っかかれる）
│  ├─ 伸長II（より高い段差を越える）
│  └─ 伸長III（擬似的なリーチ攻撃にもなる）
├─ 粘着枝
│  ├─ 粘着I（滑りにくくなる）
│  ├─ 粘着II（壁に短時間張り付ける）
│  └─ 粘着III（天井にも張り付ける）
└─ 硬化枝
   ├─ 硬化I（微小な落下耐性）
   ├─ 硬化II（接触ダメージ軽減）
   └─ 硬化III（棘が生える→接触した敵にダメージ）
```

#### Sap系（樹液）— 機動・跳躍 （約15ノード）

```
Sap
├─ 跳躍枝
│  ├─ 跳躍I（小ジャンプ解禁）
│  ├─ 跳躍II（溜めジャンプ）
│  ├─ 跳躍III（空中制御強化）
│  └─ 二段跳躍（空中でもう一度跳べる）
├─ 牽引枝
│  ├─ 牽引I（短射程グラップル解禁）
│  ├─ 牽引II（射程延長）
│  ├─ 牽引III（リール速度向上）
│  └─ 牽引IV（勢い保持+30%）
└─ 滑空枝（レア：高所のSapが必要）
   ├─ 滑空I（落下速度軽減）
   ├─ 滑空II（横方向の滑空制御）
   └─ 滑空III（上昇気流に乗れる）
```

#### Metal系（鉄粉）— 耐久・防御 （約12ノード）

```
Metal
├─ 外殻枝
│  ├─ 外殻I（最大HP+20）
│  ├─ 外殻II（落下ダメージ-30%）
│  └─ 外殻III（最大HP+50、被ダメ-20%）
├─ 重量枝
│  ├─ 重量I（ノックバック耐性）
│  ├─ 重量II（地面に根を張る＝強風耐性）
│  └─ 重量III（体当たりの威力+）
└─ 磁力枝
   ├─ 磁力I（鉄面に吸着できる）
   ├─ 磁力II（鉄片を引き寄せる＝遠距離回収）
   └─ 磁力III（磁力フィールドで小型敵を弾く）
```

#### Flesh系（肉）— 攻撃・捕食 （約12ノード）

```
Flesh
├─ 攻撃枝
│  ├─ 攻撃I（追い払い攻撃解禁：ノックバック）
│  ├─ 攻撃II（攻撃範囲拡大）
│  └─ 攻撃III（クールダウン短縮）
├─ 捕食枝
│  ├─ 捕食I（敵を倒すとFlesh確定ドロップ）
│  ├─ 捕食II（HP吸収：敵を倒すと回復）
│  └─ 捕食III（敵の能力を一時取得）
└─ 威嚇枝
   ├─ 威嚇I（周囲の小型敵が逃げる）
   ├─ 威嚇II（威嚇範囲拡大）
   └─ 威嚇III（ボス級以外は近寄らない）
```

#### 混合進化（食性の組み合わせで発生）

| 組み合わせ | 進化 | 効果 |
|------------|------|------|
| Dust + Sap | **蔓の触手** | 粘着グラップル（壁に張り付くロープ） |
| Dust + Metal | **結晶殻** | 硬い外殻+段差に引っかかりやすい |
| Sap + Metal | **磁力射出** | グラップルが鉄面に自動吸着 |
| Sap + Flesh | **跳躍捕食** | ジャンプ中の攻撃が強化 |
| Metal + Flesh | **鉄の顎** | 体当たりが攻撃になる |
| Dust + Sap + Metal | **適応殻** | 全環境に微耐性（レア） |

> 混合進化は各食性を一定比率以上摂取した場合にのみ発生する。意図的に食べ分けるプレイが必要。

## 8. プレイヤーシステム

### 8.1 初期状態

- **形状**: 球体
- **移動**: 左右移動のみ（転がり）。ジャンプ不可
- **能力**: 餌への接触吸収のみ
- **越えられる段差**: 自身の半径程度の微段差のみ

> 最初の1分で塵を食べ、伸長Iを獲得→段差を越えられるようになる体験が「成長の気持ちよさ」を伝える。

### 8.2 HP

| 要素 | 仕様 |
|------|------|
| **基礎HP** | 100（Metal進化で増加） |
| **回復手段** | 回復ポイント（希少）、捕食進化によるHP吸収 |
| **落下ダメージ** | 距離に応じて増加（小:5-10 / 中:20-35 / 大:50-100） |
| **死亡条件** | HP 0 または即死ゾーン（マップ底辺）への落下 |

### 8.3 欠片（フラグメント）システム

落下ダメージを受けると、プレイヤーの**現在の支配的食性に対応した欠片**が周囲に散らばる。

| 要素 | 仕様 |
|------|------|
| **ドロップ条件** | 落下ダメージ発生時（ダメージ量に比例して数が増える） |
| **プレイヤーが回収** | 進化ポイントとして少量回収（元の摂取量の一部が戻る） |
| **敵が回収** | 敵が欠片を食べると**敵が強化**される（移動速度UP、攻撃力UP等） |
| **消滅時間** | 一定時間で消滅（回収猶予あり） |

> 落下は「高さを失う+HPが減る」だけでなく**「敵を強化する餌を撒く」**行為になる。配信者のリアクション＆戦略的緊張感。

### 8.4 見た目の変化

進化に伴いプレイヤーの外見が変化する（段階的に実装）。

- **Phase 1（プロト）**: 食性に応じた体色変化のみ
- **Phase 2**: 進化ノードに応じたパーツ追加（手・足・翼・殻等のシルエット変化）
- **Phase 3**: 混合進化時のユニークな外見パターン

## 9. 敵システム

### 9.1 設計方針

敵の役割は**「登攀の邪魔役」**。戦闘がメインにならないよう、攻撃は「倒す」より**「追い払う・通過する」**ための手段とする。

- 攻撃は**クールダウン付きの1ボタン**（Flesh進化で解禁）
- 効果は「ノックバック」「怯み」が中心
- 敵を倒すと素材（Flesh）をドロップするが、必須ではない

### 9.2 敵タイプ一覧

#### プロトタイプ（最初に実装する3種）

| タイプ | 行動 | 脅威度 |
|--------|------|--------|
| **這い虫（追い立て役）** | プレイヤーに向かって移動、接触でダメージ。欠片を食べて加速 | 低 |
| **糸切り虫（縄切り役）** | グラップルロープに触れると一時無効化 | 中 |
| **擬態虫（フック妨害役）** | フックポイントに擬態。掴むとダメージ+落下 | 中 |

#### 拡張（後のフェーズで追加）

| タイプ | 行動 | 脅威度 |
|--------|------|--------|
| **飛行種** | 空中を巡回。滑空中のプレイヤーを妨害 | 中 |
| **重量種** | 通路を塞ぐ。ノックバック耐性が高い。体当たりで崩せる | 高 |
| **群体種** | 個体は弱いが群れで行動。一気に欠片を食い尽くす | 中 |
| **捕食者（ボス級）** | 特定エリアに出現。欠片を大量に食べて巨大化 | 最高 |
| **寄生種** | プレイヤーに付着し、移動速度を低下させる | 低〜中 |

### 9.3 敵と欠片の相互作用

- 敵が欠片を食べると**段階的に強化**される（見た目も変化）
- 強化段階: 通常 → 活性化（速度+30%）→ 凶暴化（攻撃力+50%、体が大きくなる）
- 同じ欠片でも敵の種類によって強化の仕方が異なる

## 10. マップシステム

### 10.1 構造

**セクションランダム**方式。手作りのセクション（部屋）をランダムに接続して1ランのマップを生成する。

- **方向**: 縦横ハイブリッド（メトロイドヴァニア的）
- **セクション**: 事前に手作りした部屋をプールから選び接続
- **接続ルール**: 入口/出口の位置・サイズが合うセクション同士を接続

### 10.2 バイオーム

マップは「バイオーム × 難易度」の二軸で構成される。深部ほど高難易度かつ希少な餌が出現する。

| バイオーム | テーマ | 主な餌 | 環境特徴 |
|------------|--------|--------|----------|
| **胞子の浅瀬** | 原始の温かいスープ。微生物的 | Dust（豊富） | 安全、段差が低い、チュートリアル的 |
| **樹液の柱林** | 巨大な有機構造体の林。ファンタジー的 | Sap（柱上に配置） | 垂直構造が多い、高所報酬 |
| **鉄錆の断崖** | 風化した金属質の地形。抽象的 | Metal（壁面に埋没） | 断崖・狭路、落下リスク大 |
| **捕食者の領域** | 肉食生態系の中心。原始生命的 | Flesh（敵ドロップ） | 敵が密集、戦闘必須エリア |
| **混沌の尖塔** | 全バイオームが融合した最終域 | 全種混在 | 高難易度、混合進化の機会 |

### 10.3 バイオーム × 難易度マトリクス

```
        序盤(Tier1)    中盤(Tier2)    終盤(Tier3)
胞子      ★☆☆         ★★☆          ─
柱林      ★☆☆         ★★☆          ★★★
断崖      ─            ★★☆          ★★★
捕食域    ─            ★★☆          ★★★
混沌      ─            ─             ★★★
```

> 序盤は胞子・柱林のTier1のみ。進化が進むと高Tierのセクションに到達でき、新しいバイオームが開放される。

### 10.4 マップ内の3ルート

進化の方向性に応じて、異なるルートが有利になる。

- **安定ルート**: 伸長+粘着（Dust）で堅実に登る。壁面を這い進む
- **機動ルート**: 跳躍+グラップル（Sap）で一気に高度を稼ぐ。ショートカット可能だが操作難度高
- **突破ルート**: 耐久+攻撃（Metal/Flesh）で敵を排除しながら進む。戦闘的

## 11. ゴールシステム

### 11.1 3ゴール方式

「どこかのゴールに辿り着けばクリア」。ゴールごとに要求される進化が異なり、自然にリプレイ性が生まれる。

#### G1: 浅瀬の光（The Shallow Light）

- **位置**: マップ中層の比較的到達しやすい場所
- **必要な進化**: Dust/Sapの基本進化で到達可能
- **難易度**: ★★☆（初クリア用）
- **体験**: 「初めてクリアできた！」の達成感。ゲームの全体像を理解する
- **配信的価値**: 視聴者が攻略法を一緒に考えられる、到達可能性が見える

#### G2: 天頂（The Zenith）

- **位置**: マップ最上部。垂直方向の到達限界
- **必要な進化**: 高度な機動力（Sap上位：グラップル+跳躍+滑空）
- **難易度**: ★★★（正統派クリア）
- **体験**: 長い垂直登攀の末の達成。一度の落下で大量の進捗を失うリスク
- **配信的価値**: 「あと少し！」からの落下＝最高の配信映え

#### G3: 根源の核（The Primordial Core）

- **位置**: マップ深部の横方向分岐。捕食者の領域の奥
- **必要な進化**: 戦闘能力（Flesh上位）+ 耐久（Metal）
- **難易度**: ★★★★（裏ゴール）
- **体験**: 探索と戦闘で道を切り開く。「こんな場所があったのか」という発見
- **配信的価値**: 存在自体がネタバレ要素。発見した瞬間の驚き

### 11.2 ゴールと食性の対応

```
              浅瀬の光(G1)
             ╱
    Dust/Sap基本
   ╱                  天頂(G2)
  ╱                  ╱
START ─── Sap特化 ──╱
  ╲
   ╲              根源の核(G3)
    ╲            ╱
    Flesh/Metal ╱
```

### 11.3 クリア後のやりこみ要素

- **全ゴール制覇**: 3ゴールすべてをクリアする
- **図鑑コンプリート**: 50+ノードの系統樹を全発見
- **混合進化の発見**: 特殊な食べ合わせで発生するレア進化の発見
- **タイムアタック**: 各ゴールの最速クリア
- **縛りプレイ**: 単一食性クリア、無進化クリア等

## 12. 図鑑（永続システム）

### 12.1 進化図鑑

- 発見した進化ノードが**系統樹として永続記録**される
- 4つの大枝（食性）が視覚的に分かれた樹形図として表示
- 未発見ノードは「???」表示（存在のヒントだけ見える）
- **図鑑による恒久強化はなし**（難易度の価値を維持）

### 12.2 記録される統計

- 総ラン数 / 各ゴールクリア数
- ゴール別ベストHP / ベストタイム

## 14. 世界観・ビジュアル

### 14.1 世界設定

原始の海から始まる生命の進化を、ミクロとマクロの境界が曖昧な異世界として描く。現実の生物学に緩く基づきつつ、ファンタジー的な自由さを持つ。

- **胞子の浅瀬**: 温かい原始スープ。有機物の粒子が浮遊する柔らかい環境
- **樹液の柱林**: 巨大な有機構造体が林立。細胞壁のような半透明の柱
- **鉄錆の断崖**: 金属酸化物の地形。幾何学的で抽象的な美しさ
- **捕食者の領域**: 生態系の頂点。脈打つ肉壁と骨格構造
- **混沌の尖塔**: 全環境が融合。生命の多様性を凝縮した最終域

### 14.2 ビジュアル方針

- **初期実装**: Phaser生成テクスチャ（プリミティブ形状＋カラーリング）
- **方向性**: 抽象的かつ有機的。明確なシルエットで視認性を確保
- **色彩設計**: バイオームごとに明確なカラーパレットを持つ

## 19. リスクと対策

| リスク | 影響度 | 対策 |
|--------|--------|------|
| **「球体で何もできない」初期体験が退屈** | 高 | 最初のDust餌を30秒以内に配置。即座に成長を実感させる |
| **食性自動進化が分かりにくい** | 高 | 進化時のエフェクト+UI表示で「何を食べたから何になった」を明示 |
| **セクションランダムで理不尽な配置** | 中 | セクション設計時に最低限の到達保証を入れる。Dustルートは常に確保 |
| **敵が登攀の邪魔になりすぎる** | 高 | 敵密度を低めに設定。捕食者の領域以外では回避可能に |
| **50+ノードのバランス崩壊** | 中 | ノード追加は段階的に。各プロトでバランステスト |
| **混合進化の組み合わせ爆発** | 中 | 混合進化は計画的に設計。全組み合わせではなく厳選したもののみ |
