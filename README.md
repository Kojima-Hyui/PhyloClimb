# PhyloClimb

グラップル登攀アクション × ラン制進化 × 進化図鑑

## 概要

「Getting Over It」的な緊張感とローグライク的な進化システムを融合した、配信向け高難易度アクションゲーム。

## 技術スタック

- **エンジン**: Phaser3 (予定)
- **言語**: TypeScript
- **ターゲット**: Steam (PC)

## 開発状況

Phase 1: プロトタイプ開発中

## ドキュメント

- [企画書](./企画書.md) - 詳細なゲームデザインドキュメント

